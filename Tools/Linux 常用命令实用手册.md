本文按功能分类整理Linux高频命令，标注核心参数与使用场景，适合日常运维、开发调试及新手入门，所有命令均基于主流Linux发行版（Ubuntu、CentOS等）通用语法。

# 一、文件与目录操作（最基础高频）

## 1.1 目录导航

- `pwd`：打印当前工作目录路径（显示所在位置的绝对路径）。
    
- `cd 目录路径`：切换工作目录。
    
    - `cd ~`：切换到当前用户主目录（等价于 `cd /home/用户名`）。
        
    - `cd ..`：切换到上一级目录。
        
    - `cd -`：切换到上一次所在的目录。
        
- `ls`：列出当前目录内容（默认显示非隐藏文件）。
    
    - `ls -l`（简写 `ll`）：以长格式显示，包含权限、所有者、大小、修改时间等信息。
        
    - `ls -a`：显示所有文件（包括隐藏文件，以 `.` 开头的文件）。
        
    - `ls -lh`：长格式显示，大小以KB/MB/GB友好单位展示。
        

## 1.2 文件/目录创建与删除

- `touch 文件名`：创建空文件，若文件已存在则更新其修改时间。
    
- `mkdir 目录名`：创建新目录。
    
    - `mkdir -p 目录路径`：递归创建多级目录（如 `mkdir -p /a/b/c`，无需手动创建a、b目录）。
        
- `rm 文件名/目录名`：删除文件或目录（默认不删除目录）。
    
    - `rm -f 文件名`：强制删除文件，忽略不存在的文件，无提示。
        
    - `rm -r 目录名`：递归删除目录及目录内所有内容。
        
    - `rm -rf 目录名`：强制递归删除目录（高危命令，慎用，避免误删系统文件）。
        

## 1.3 文件/目录复制与移动

- `cp 源文件 目标路径`：复制文件。
    
    - `cp -r 源目录 目标路径`：递归复制目录及内容。
        
    - `cp -p 源文件 目标路径`：复制时保留文件原有权限、修改时间等属性。
        
- `mv 源文件/目录 目标路径`：移动文件/目录，也可用于重命名。
    
    - 重命名：`mv 旧文件名 新文件名`（同一目录下操作）。
        
    - 跨目录移动：`mv /a/file /b/`（将/a目录下的file移动到/b目录）。
        

## 1.4 文件内容查看

- `cat 文件名`：一次性显示文件全部内容（适合小文件）。
    
    - `cat -n 文件名`：显示内容并添加行号。
        
- `more 文件名`：分页显示文件内容，按空格键翻页，按`q`退出。
    
- `less 文件名`：分页显示文件内容，支持上下键滚动、搜索，按`q`退出（功能比more更强大）。
    
- `head 文件名`：显示文件前10行内容。
    
    - `head -n 行数 文件名`：显示指定前N行（如 `head -n 20 test.txt` 显示前20行）。
        
- `tail 文件名`：显示文件后10行内容。
    
    - `tail -n 行数 文件名`：显示指定后N行。
        
    - `tail -f 文件名`：实时监控文件内容变化（常用于查看日志，按`Ctrl+C`退出）。
        

# 二、系统管理与监控

## 2.1 系统信息查看

- `uname -a`：显示系统内核版本、主机名、硬件架构等完整信息。
    
- `hostname`：显示当前主机名，`hostname 新主机名` 临时修改主机名（重启失效）。
    
- `date`：显示当前系统时间与日期，`date -s "2026-01-27 10:30:00"` 手动修改时间。
    
- `df -h`：查看磁盘分区使用情况，以友好单位展示容量、已用、剩余空间。
    
- `du -sh 目录名`：查看指定目录总大小（`-s` 汇总，`-h` 友好单位）。
    

## 2.2 进程管理

- `ps`：查看当前终端进程。
    
    - `ps -ef`：显示系统所有进程（完整格式，包含PID、父进程ID、所有者等）。
        
    - `ps aux`：显示所有进程及资源占用情况（常用）。
        
- `top`：实时监控进程资源占用，按CPU、内存排序（按`q`退出，按`P`按CPU排序，按`M`按内存排序）。
    
- `kill 进程PID`：终止指定PID的进程（PID可通过ps、top查看）。
    
    - `kill -9 进程PID`：强制终止进程（适用于常规kill无法终止的进程）。
        
- `pkill 进程名`：按进程名终止进程（如 `pkill firefox` 终止所有火狐浏览器进程）。
    

## 2.3 服务管理

- Systemd系统（主流发行版默认，如Ubuntu 16+、CentOS 7+）：
    
    - `systemctl start 服务名`：启动服务（如 `systemctl start nginx`）。
        
    - `systemctl stop 服务名`：停止服务。
        
    - `systemctl restart 服务名`：重启服务。
        
    - `systemctl status 服务名`：查看服务运行状态。
        
    - `systemctl enable 服务名`：设置服务开机自启。
        
    - `systemctl disable 服务名`：取消服务开机自启。
        
- SysVinit系统（旧版发行版，如CentOS 6）：
    
    - `service 服务名 start/stop/restart/status`：服务启停与状态查看。
        

# 三、用户与权限管理

## 3.1 用户管理

- `useradd 用户名`：创建新用户（默认无密码，需设置密码才能登录）。
    
- `passwd 用户名`：设置/修改用户密码（普通用户可修改自身密码，root用户可修改所有用户密码）。
    
- `userdel 用户名`：删除用户。
    
    - `userdel -r 用户名`：删除用户同时删除其主目录。
        
- `su - 用户名`：切换到指定用户（`-` 表示加载目标用户的环境变量）。
    
- `sudo 命令`：以root权限执行指定命令（需用户在sudoers配置文件中授权）。
    

## 3.2 权限管理

Linux文件权限分为读（r=4）、写（w=2）、执行（x=1），分别对应所有者（u）、所属组（g）、其他用户（o）三类对象，权限格式如`rwxr-xr--`。

- `chmod 权限值 文件名/目录名`：修改文件/目录权限。
    
    - 数字法：`chmod 755 test.sh`（所有者rwx，所属组r-x，其他用户r-x，7=4+2+1，5=4+1）。
        
    - 符号法：`chmod u+x test.sh`（给所有者添加执行权限）、`chmod o-w test.txt`（取消其他用户写权限）。
        
- `chown 所有者:所属组 文件名/目录名`：修改文件/目录的所有者和所属组。
    
    - `chown root:root test.txt`：将test.txt的所有者和所属组都改为root。
        
    - `chown -R 用户名:组名 目录名`：递归修改目录及内容的所有者和所属组。
        

# 四、网络管理

- `ifconfig`：查看/配置网络接口信息（旧版命令，部分系统需安装net-tools工具）。
    
- `ip addr`：查看网络接口IP地址等信息（新版推荐命令，替代ifconfig）。
    
- `ping 目标IP/域名`：测试与目标主机的网络连通性（按`Ctrl+C`退出）。
    
    - `ping -c 4 目标IP`：发送4个数据包后停止（避免无限循环）。
        
- `curl 网址`：发送HTTP请求，获取网页内容或测试接口（如 `curl https://www.baidu.com`）。
    
- `wget 下载链接`：从网络下载文件（如 `wget https://xxx.com/file.tar.gz`）。
    
- `netstat -tuln`：查看系统监听的端口号（`-t` tcp，`-u` udp，`-l` 监听，`-n` 数字显示IP和端口）。
    

# 五、压缩与解压

## 5.1 常见压缩格式

- tar格式（打包，可结合gzip/bzip2压缩）：
    
    - 打包压缩：`tar -zcvf 压缩包名.tar.gz 源文件/目录`（gzip压缩，最常用）、`tar -jcvf 压缩包名.tar.bz2 源文件/目录`（bzip2压缩）。
        
    - 解压：`tar -zxvf 压缩包名.tar.gz`（解压到当前目录）、`tar -zxvf 压缩包名.tar.gz -C 目标目录`（解压到指定目录）。
        
- zip格式：
    
    - 压缩：`zip -r 压缩包名.zip 源文件/目录`（`-r` 递归压缩目录）。
        
    - 解压：`unzip 压缩包名.zip`、`unzip 压缩包名.zip -d 目标目录`（解压到指定目录）。
        
- rar格式（需安装rar工具）：
    
    - 压缩：`rar a 压缩包名.rar 源文件/目录`。
        
    - 解压：`rar x 压缩包名.rar`。
        

# 六、其他实用命令

- `clear`：清空终端屏幕（快捷键 `Ctrl+L` 效果相同）。
    
- `history`：查看历史执行过的命令，`!命令序号` 可重复执行对应命令（如 `!100` 执行第100条历史命令）。
    
- `find 查找路径 -name "文件名"`：按文件名查找文件（如 `find /home -name "test*.txt"` 查找/home目录下所有以test开头、.txt结尾的文件）。
    
- `grep "关键词" 文件名`：在文件中搜索包含关键词的行（如 `grep "error" log.txt` 查找日志中的错误信息）。
    
    - `grep -r "关键词" 目录名`：递归搜索目录下所有文件中的关键词。
        
- `shutdown -h now`：立即关机，`shutdown -r now`：立即重启，`reboot`：快速重启。
